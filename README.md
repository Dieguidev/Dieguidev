### 🌟 Proyectos Destacados

Aquí hay algunos de mis proyectos más destacados:

- **biAnalytics**: biAnalytics es un proyecto desarrollado con Next.js, diseñado para proporcionar análisis de datos de manera eficiente. Utiliza TypeScript como lenguaje principal y está configurado para ser desplegado fácilmente en la plataforma Vercel.
  - **Tecnologías**: Next.js, TypeScript, CSS, JavaScript.
  - **Características**:
    - Configuración inicial con `create-next-app`.
    - Optimización automática de fuentes con `next/font`.
    - Despliegue sencillo en Vercel.
  - **Enlace al repositorio**: [biAnalytics](https://github.com/Dieguidev/biAnalytics)
- **Dieguidev Shop**: Dieguidev Shop es una aplicación e-commerce moderna, desarrollada con Next.js y TypeScript. Incluye gestión de productos, carrito, checkout, autenticación, integración con Prisma y una UI profesional y responsiva.
  - **Tecnologías**: Next.js, TypeScript, Prisma ORM, Tailwind CSS, Zustand, Swiper.js.
  - **Características**:
    - Gestión de productos, carrito de compras y proceso de checkout.
    - Autenticación y registro de usuarios.
    - Optimización de imágenes con `next/image`.
    - Uso de `docker-compose` para entornos de desarrollo con PostgreSQL.
  - **Enlace al repositorio**: [Dieguidev Shop](https://github.com/Dieguidev/next-dieguidev-shop)
- **Nest-GPT**: Nest-GPT es una API RESTful desarrollada con NestJS y TypeScript que utiliza la inteligencia artificial de OpenAI para corregir textos, analizar pros y contras, traducir textos y convertir texto a audio.
  - **Tecnologías**: NestJS, TypeScript, OpenAI GPT-3.5 Turbo.
  - **Características**:
    - Corrección ortográfica y gramatical avanzada.
    - Análisis de pros y contras en formato markdown.
    - Traducción de textos a cualquier idioma.
    - Síntesis de voz con múltiples opciones de voces.
    - Endpoints con soporte para streaming.
  - **Enlace al repositorio**: [Nest-GPT](https://github.com/Dieguidev/nest-openia)
